# 🧠 Data Mining Project: Python Job Market in Egypt

This project is a complete Data Mining pipeline that scrapes, cleans, analyzes, and visualizes Python job listings in Egypt.

---

## 📌 Project Overview

- **Web Scraping**: Data collected from **Wuzzuf** and **Tanqeeb** using Python (BeautifulSoup & Requests).
- **Data Cleaning**: Handled missing values, standardized location names, removed duplicates, and dropped unnecessary columns.
- **Database**: Stored cleaned data in **SQLite** database for querying.
- **Analysis**: SQL queries to extract insights (top jobs, top companies, location distribution).
- **Visualization**: Created 6 visualizations using **Matplotlib** & **Seaborn** to tell the data story.

---

## 🔍 Key Insights

- **Most In-Demand Job Title**: Senior Data Engineer  
- **Top Hiring Company**: Joveo AI  
- **Dominant Location**: Cairo (81.1%)  
- **Top Job Category**: AI / ML (104 listings)  
- **Employment Type**: Full-time jobs dominate the market.
